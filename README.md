# MacBook Pro Landing Page

A responsive landing page inspired by Apple's MacBook Pro website.

## 🔧 Technologies Used

* HTML5
* CSS3 (Flexbox, responsive design)
* Git & GitHub

## 🎯 Features

* Clean, modern layout inspired by a real-world product page
* Responsive design across desktop and mobile
* Structured navigation and header layout
* Reusable CSS styling patterns
